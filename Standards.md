# Add to 20 Interactive

This repository contains the code for the **Add to 20 Interactive**, designed to help **elementary students** (Grades 1–2) practice **basic addition within 20** using engaging visuals and number models. By combining numbers and images, learners strengthen their fluency with addition and build a foundation for mental math strategies.
